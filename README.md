### 프로젝트 설치 및 실행

#### 1.`git clone https://github.com/kjb9301/tp-test.git`

    > 원하는 디렉토리에서 위 명령어를 통해 해당 프로젝트를 가져옵니다.

#### 2.`npm install`

    > 가져온 tp-test 디렉토리로 이동.
    > 이동 후, 위 명령어를 통해 프로젝트에 사용된 모듈을 설치.

#### 3.`npm start`

    > 위 명령어를 통해 App을 실행.
    > ※ **크롬(Chrome)**에서 [http://localhost:3000]을 열어주세요.
